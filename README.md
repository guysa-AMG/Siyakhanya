## ðŸ“Š Best Use of Data & Logic

We use advanced modeling to overcome data scarcity and provide predictive insights, demonstrating the work of a **Data Analyst**.

### 1\. The Raw Data Advantage

  * **Grid Risk Data:** We use **raw Historical UCLF data** (Unplanned Capability Loss Factor) to quantify the national grid's historical instability. This data directly feeds the **RAP** calculation.
  * **Affordability Data:** We use **NERSA tariff structure** and community income data to define the **Affordability Ceiling** for the Community Tool.

### 2\. The ML Logic

  * **Advanced Calculation:** The tool uses simple **Machine Learning (ML) logic** (Time-Series Weighting) to predict the **hourly demand profile** for both markets, which is crucial for correctly sizing the battery and solar system.
