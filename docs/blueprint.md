# **App Name**: siyaKhanyaAI

## Core Features:

- Community Selection: Allow users to select a community to view energy predictions and payback scenarios.
- Hourly Demand Prediction: Display a chart of predicted hourly energy demand for the selected community using energy forecasting models.
- Payback Scenario Builder: Allow users to input system size, battery capacity, and grid price to calculate payback.
- Payback Calculation: Calculate the payback period and NPV (Net Present Value) for the specified system configuration using discount rate.
- Consumption Data Upload: Allow users to upload their consumption data via CSV files.
- Anomaly Detection in Consumption Data: Use a generative AI tool to detect anomalies in uploaded consumption data, identifying unusual energy usage patterns and notifying the user.

## Style Guidelines:

- Primary color: Teal green (#0FB69A) for a friendly and energetic feel.
- Accent color: Warm sunset orange (#FF7A59) to highlight key actions and alerts.
- Background color: Soft off-white (#F6F7F9) for a clean and muted backdrop.
- Text color: Deep charcoal (#222831) for readability.
- Body and headline font: 'Inter' (sans-serif) for a clean, modern, neutral feel.
- Use simple, outlined icons to represent different energy-related concepts.
- Divide the screen into a left panel for the scenario builder and a right panel for the consumption chart.
